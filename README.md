# scpFolderSender

## CONFIG
Créez un fichier config.json avec le contenu suivant :

{
    "server_address": "YourIP",
    "username": "YourUsername",
    "password": "YourPassword"
}

## LAUNCH
Sélectionnez le dossier à envoyer et le type de données.

## AMELIORATION
